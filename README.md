# Setup env node

Windows
```
$ set NODE_OPTIONS=--openssl-legacy-provider
```
Linux
```
$ export NODE_OPTIONS=--openssl-legacy-provider
```

# Install dep
```
$ cd AI-translator
$ pnpm install
```
## Start APP
```
$ npm start
```

# Run Build
```
$ pnpm run build
$ pnpm install -g serve
$ serve -s build
```
